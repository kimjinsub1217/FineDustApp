
# 📌 FineDustApp
> 미세먼지 앱

## ✔기능
<ul>
  <li> 내 위치 정보를 가져올 수 있다.  </li>
  <li> 공공데이터 에어 코리아 API를 통해 미세먼지 정보를 가져올 수 있다. </li>
  <li> 미세먼지 위젯을 추가할 수 있다.</li>
</ul>

##  ✔활용 기술
LocationManager, Retrofit2, Coroutine, App Widgets
